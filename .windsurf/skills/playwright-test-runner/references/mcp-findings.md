# MCP Findings

Document selector discoveries and UI patterns found via MCP browser inspection.

## How to Use This File

When using MCP browser to discover selectors:
1. Document the element and its ref/selector
2. Note any nth() indices
3. Record state transitions (what changes after interactions)
